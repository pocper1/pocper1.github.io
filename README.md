# My Portfolio

A minimalist portfolio website built with Next.js and TypeScript.

## Tech Stack

-   Next.js 14 (App Router)
-   TypeScript
-   Tailwind CSS
-   shadcn/ui
-   pnpm

## Getting Started

### Prerequisites

-   Node.js 18+
-   pnpm 8+

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Customization

Edit [src/data/resume-data.tsx](./src/data/resume-data.tsx) to update your personal information, work experience, education, and skills.
